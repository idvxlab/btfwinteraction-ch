为了探索BTFW交互设计模式的使用，我们对109名参与者进行了一次用户研究。

基于初步研究结果，我们从四个方面来衡量设计模式，包括自我故事联系、用户参与、信息回忆、以及互动性和可理解性之间的平衡。为了测量自我故事的联系，笔者参考了在市场营销领域用于评估自我品牌联系的工具[1]。笔者认为这是一个合适的工具，因为自我故事和自我品牌联系都侧重于将叙事纳入人们的自我概念中。笔者修改了[1]所提出的问题，以适应数据故事的背景。例如，"X品牌反映我是谁 "被改写为 "这个故事反映我是谁"。因此，笔者的问卷中使用了四个与自我故事联系相关的项目，包括反映、联系、识别和匹配。在用户参与方面，笔者沿用了[2]中使用的方法，从数据驱动的故事角度评估参与度。具体来说，笔者的问卷中使用了七个与用户参与有关的项目，包括集中注意力、享受、兴趣、感觉参与、挑战、动机和喜欢。在笔者的问卷中，关于自我故事连接和用户参与的项目都使用了7点李克特量表（1表示 “完全不同意”，7表示 “完全同意”）。信息回忆是通过在用户研究的两周后对参与者进行在线访谈来测量的。他们被要求描述他们对所读故事的记忆。此外，为了测量互动性和可理解性之间的平衡，笔者指示参与者按照Lan等人[3]提出的方法，在网格上对一个数据故事的互动性和可理解性进行评分。在网格中，X轴代表互动性的尺度，而Y轴代表可理解性的尺度，两者的范围是-3到3。

为了对用户研究结果进行统计分析，我们首先进行了Shapiro-Wilk检验，发现定量数据不是正态分布。因此，使用Wilcoxon Signed-Rank检验来比较基线故事 (C0) 和互动故事 (C1-C6) 作为一组，用Kruskal-Wallis H检验来进行 (C1-C6) 之间的相互比较。

自我故事的连接

Wilcoxon Signed-Rank检验结果显示，在自我故事连接方面，有BTFW互动的故事比基线故事的得分要高。通过分别观察自我故事联系的四个指标，有BTFW互动的故事在所有指标上都获得了明显的高分，包括反映（z= -5.606， p < .001）、联系（z = -2.934， p = .003）、识别（z = -3.521， p < .001）和匹配（z = -2.928， p = .003）。这一结果与笔者在访谈中发现的一致，即大多数参与者都认为BTFW互动促进了他们与故事建立联系（例如，“这就像一个净化的过程，过滤掉与我无关的叙述”（P18））。然后，为了考察六种模式中自我故事联系的差异笔者进行了Kruskal-Wallis H检验。根据结果，六种设计模式在反映（H（5） = 22.641， p < .00）、联系（H（5）=12.284，p=.031）和识别（H（5）=12.696，p =.026）方面有明显的差异。Dunn的事后检验表明，C1在反映、联系和匹配方面得分最高（见图6.2）。具体来说，在反思方面，C1明显高于C3（p = .001）和C4（p = 001）;在联系方面，C1明显高于C3（p = .023）;在认同方面，C1获得的分数明显高于C4（p = .026）。原因可能是在开始时产生的联系感会在整个故事中一直伴随着参与者（例如，“当我意识到下面的内容是为了解释我个性化的开始，我觉得它与我更加相关”（P7））。

用户参与

根据Wilcoxon Signed-Rank检验的结果，C1-C6的七项指标在参与度方面的得分都明显高于C0。具体来说，这七个指标包括集中注意力（z = -4.929，p < .001），享受（z=-4.000，p< .001），兴趣（z = -4.454，p<.001），参与感（z=-5.403，p<.001），挑战（z=-3.492，p< .001），动机（z = -4.344，p < .001）和喜爱程度（z =-4.028，p<.001）。定性反馈显示，许多参与者将他们更深的参与归因于更强的自我故事联系（例如，“当故事专门讲给我听时，我会更仔细地阅读它”（P23））。然后，笔者进行了Kruskal-Wallis H检验，在参与度方面相互比较这六种模式。结果显示这六种模式之间没有明显的差异。如图6.2所示，在这些模式中，C1在兴趣（M=5.89）、参与感（M=5.62）和好感度（M=5.43）方面的平均值最高，C2在挑战方面的平均值最高（M=5.56），而C5在集中注意力（M=5.83）享受（M=5.42）和动机（M=5.61）方面的平均值最高。在访谈中，参与度高的原因是多样的。有些人解释说这是因为享受（例如，“我很惊讶我答对了七道题，我很高兴”（P4），其他人提到了集中注意力（例如，“我在C1中投入了很多注意力来寻找自己的疫苗线”（P10））和动机（例如“阅读C5就像参加比赛，我很想猜下一个问题”（P2））。

互动性-可理解性的平衡

互动性和可理解性之间的平衡结果显示在图2中。在互动性方面（网格中的X轴），互动故事获得的分数明显高于基线（z=-8.488，p< .001）。Kruskal-Wallis H检验证实，六种模式在互动性方面没有明显的差异（H（5）=51.190，p=.163）。虽然C6获得了最高分（M=2.04），但C2（M=1.55）和C4（M=1.49）被认为是相对低的互动性。在访谈中，一些参与者提到他们根据阅读过程中的互动频率来评价互动性（例如，“C6的大部分内容是由互动构成的（P24）），另一些参与者则解释说他们根据行动的自由度来打分（例如，“通过C3提供的三个滑块，我可以创造无数的输入方式”（P1））。对于可理解性（网格中的Y轴），互动故事和基线之间没有明显的差异（z = -1.919，p=.055）。此外，六种模式在可理解性方面也没有明显的相互差异（H（5）=2.894，p=.716）。虽然C1获得了最高分（M = 2.04），但C1-C5都被认为比基线更容易理解（M = 1.82），只有C6的理解性最低（M = 1.66）。一些参与者解释说，他们给C6的可理解性打了低分，因为作者缺乏解释（例如，“我本来以为作者会告诉我价格变化的原因，然而，他什么也没解释，在我做完所有的问题后就匆匆结束了故事”（P2））。

总之，结果表明，整合了BTFW互动的数据故事 (C1-C6) 在自我故事连接、用户参与、信息回忆和互动性-可理解性平衡方面比基线 (C0) 表现更好。
